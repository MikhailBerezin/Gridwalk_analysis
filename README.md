# Gridwalk foot-fault analysis (nerve window paper, Figure 1h)

This repository contains the custom Python script and source data used to
generate **Figure 1h** of the manuscript "Longitudinal Single-Axon-Resolution
Imaging of Peripheral Nerve Injury Response in Mice Using an Optical Window
Implant" (Luzhansky et al., *Advanced Science*).

The script loads gridwalk foot-fault scoring data from an Excel workbook,
computes the per-bin mean ± SEM weighted success rate for the windowed and
contralateral hindlimb, and produces the bar plot shown in Figure 1h.

Statistical comparisons reported in the paper (paired two-tailed t-test and
Wilcoxon signed-rank test) can be reproduced from the loaded arrays using
standard `scipy.stats` calls — see "Reproducing the statistics" below.

## Files

| File | Description |
|------|-------------|
| `gridwalk_analysis.py` | Python script that loads the data and produces Figure 1h |
| `gridwalk_data.xlsx`   | Source data: 21 paired (windowed vs contralateral) gridwalk trials from 8 mice across 5 post-implantation time bins, plus one trial at day 81 |
| `figure_1h.png`        | Output plot generated by the script |
| `README.md`            | This file |

## Data format

The Excel workbook contains one row per gridwalk trial. The script reads the
following columns:

| Column | Field |
|--------|-------|
| A | Mouse identifier |
| C | Foot tested ("L"/"left" = contralateral, "R"/"right" = windowed) |
| E | Days after implant (negative = pre-implantation) |
| K, L, M, N | Counts of "G" / "M" / "P" / "B" steps (letter-based scoring; older trials) |
| O, P, Q, R, S | Counts of digits 0 / 1 / 2 / 3 / 4 (digit-based scoring; newer trials) |

Each trial uses **either** the letter-based or digit-based count columns,
not both. The script auto-detects which is present.

## Scoring

Each foot placement on the wire grid was scored on a 0-1 scale, where
**1 = clean step** (foot placed firmly on the wire) and **0 = complete miss**
(foot fell through the grid opening). Two scoring conventions were used over
the course of the study; both produce a weighted success rate on the same
0-1 scale.

**Letter-based (3-level, used in earlier trials):**

| Letter | Description | Score |
|--------|-------------|------:|
| G | Good step (clean) | 1.00 |
| M | Medium (partial slip with weight shift) | 0.50 |
| P | Poor step | 0.00 |
| B | Bad step (missed) | 0.00 |

Weighted success = (G + 0.5·M) / (G + M + P + B)

**Digit-based (5-level, used in later trials):**

| Digit | Description | Score |
|-------|-------------|------:|
| 4 | Clean step (foot on wire, full grip) | 1.00 |
| 3 | Foot on wire with slight adjustment | 0.75 |
| 2 | Partial slip, foot recovered on wire | 0.50 |
| 1 | Deep slip with partial recovery | 0.25 |
| 0 | Complete miss (fell through grid) | 0.00 |

Weighted success = (4·n4 + 3·n3 + 2·n2 + 1·n1 + 0·n0) / (4·N) = (n4 + 0.75·n3 + 0.5·n2 + 0.25·n1) / N

The success rate per trial is reported as a percentage.

## Binning

Trials are binned into five post-implantation intervals:

| Bin label | Days range |
|-----------|-----------:|
| <0   | pre-implantation |
| 1-7   | 1 to 7 |
| 8-13  | 8 to 13 |
| 14-25 | 14 to 25 |
| 26-35 | 26 to 35 |

n = 4 paired (L/R) observations per bin (8 unique mice contribute across the
bin set; not every mouse was tested at every time point).

## Reproducing the statistics

The paper reports paired two-tailed t-tests and Wilcoxon signed-rank tests
on the (right - left) differences. To reproduce these from the loaded data:

```python
from scipy import stats
from gridwalk_analysis import load
import numpy as np

days, success, side = load("gridwalk_data.xlsx")
# Pair L and R observations by (mouse_implicit_via_day, day) - here we trust
# that L and R rows are present together for each tested timepoint.
# For convenience:
L_mask = side == "L"
R_mask = side == "R"

# Paired t-test (overall, pooled across bins):
t, p = stats.ttest_rel(success[L_mask], success[R_mask])
w, p_w = stats.wilcoxon(success[L_mask], success[R_mask])
print(f"paired t p = {p:.4f}, Wilcoxon p = {p_w:.4f}")
```
